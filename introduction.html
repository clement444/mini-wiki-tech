Introduction à Git

Git est un système de gestion de versions utilisé pour suivre les modifications du code, faciliter la collaboration entre les développeurs et permettre de revenir facilement à des versions antérieures d'un projet.

Installation

  • Téléchargez et installez Git depuis le site officiel.
  • Configurez votre identité :git config --global user.name "Votre Nom" et git config --global user.email "vous@exemple.com".
  • Initialisez un dépôt dans un dossier existant avec git init ou clonez un dépôt distant avec git clone.
