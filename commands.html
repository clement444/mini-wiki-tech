Commandes Git essentielles

En git, on a plusieurs commandes qui sont essantielle pour pouvoir faire ses repository, sans elles impossible de faire quoi que se soit.

Ces commandes sont:

Init, add, commit, push, branch, merge

Commande Description
init Permet d'initialiser le repo, à utiliser à chaque fois que vous voulez créer nouveau un projet.
add Permet de mettre en mémoire vos changement/ajout de fichiers, cette commande ne permet que de préparer vos fichiers pour un commit.
commit Permet de sauvegarder vos changements sur votre branche en local
push Permet de sauvegarder vos changements sur le server distant github
branch Permet de lister les branches existantes, elle peut permettre de créer de nouvelles branches si un nom est spécifié en option
merge Permet de fusionner votre branche avec une autre
